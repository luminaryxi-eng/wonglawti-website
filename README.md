# Wong, Law & Ti — Official Website

Personal law firm website for **Kevin Goh**, Advocate & Solicitor, based in Kuantan, Pahang.

## About

This is the official landing page for **Wong, Law & Ti**, a legal practice located in Kuantan, Pahang, Malaysia. The site presents the firm's practice areas and provides a point of contact for prospective clients.

## Practice Areas

- Buy & Sale of Property (Conveyancing)
- Will & Estate
- Employment Law
- Others (General Practice)

## How to Use

This website is a single HTML file — no installation or setup required.

### View locally
Just open `index.html` in any web browser.

### Host on GitHub Pages
1. Go to your GitHub repository
2. Click **Settings** → **Pages**
3. Under *Source*, select **Deploy from a branch**
4. Choose `main` branch and `/ (root)` folder
5. Click **Save**
6. Your site will be live at `https://your-username.github.io/your-repo-name`

## Customisation

To update the website, open `index.html` in any text editor (e.g. Notepad, VS Code) and edit:

- **Your details** — search for `Kevin Goh` or `Wong, Law & Ti` to update names
- **Address / contact info** — search for `Kuantan`
- **Stats** (years of experience, cases handled) — search for the `stats-band` section
- **Practice area descriptions** — find each `pcard` block

## Tech Stack

- Plain HTML, CSS, and JavaScript — no frameworks or dependencies
- Google Fonts (Cormorant Garamond + Outfit) — loaded via CDN
- Fully mobile-responsive with hamburger menu
